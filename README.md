# bootstrap-boilerplate
This project is a Bootstrap 5.1 boilerplate template. It uses the 5.1 CDN and only uses 1 bootstap link in the head tag & 1 script tag at the end of the body tag.
The project includes 
1. index.html
2. css folder
  -styles.css
3. js folder
  -scripts.js
4. img folder
